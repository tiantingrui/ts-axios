# ts-axios

## Features
+ 在浏览器端使用 XMLHttpRequest 对象通讯
+ 支持Promise API
+ 支持请求和响应的拦截器
+ 支持请求数据和响应数据的转换
+ 支持请求的取消
+ JSON数据的自动转换
+ 客户端防止XSRF
